# OWBN Bylaws — Source Documents

Static copies of the One World by Night organizational bylaws in Markdown and HTML format.

**Status:** Reference archive. Updated periodically but not the live source of truth — bylaws are managed on council.owbn.net via the [Bylaw Clause Manager](https://github.com/One-World-By-Night/bylaw-clause-manager) plugin.

## Contents

- `Administrative/` — Administrative Bylaws (organizational governance)
- `Character/` — Character Bylaws (character creation, genre-specific rules)
- `Coordinator/` — Coordinator Bylaws (coordinator authority and procedures)

Each directory contains `.md` (Markdown) and `.html` (rendered) versions. The Character directory also has genre-specific subdirectories (Changeling, Vampire, etc.).
